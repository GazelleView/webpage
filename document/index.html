LuxeNest
Luxury Home Decor & Interior Template

Thank you for choosing LuxeNest — a sophisticated, multi‑page HTML template designed for high‑end home decor brands, furniture boutiques, artisan studios, and interior designers.

LuxeNest wraps your collection in warmth, texture, and quiet luxury. It comes with 15+ production‑ready pages: an enriched home page, a filterable shop, immersive galleries with a lightbox, a blog, detailed legal/support sections, and a design‑concierge contact flow. Every element is crafted to feel like a slow exhale.

Version: 1.0 | Last Updated: May 2026

🚀 Installation

LuxeNest uses plain HTML, Tailwind CSS via CDN, and vanilla JavaScript. No build tools, npm, or package managers are required.

1. Download and unzip the purchased package. 2. Open the project folder. 3. Double‑click index.html to preview in your browser. 4. For editing, we recommend VS Code with the Live Server extension.
Quick Start: Just open index.html and start editing. All dependencies load from CDNs — no command line needed.

  • Tailwind CSS (CDN) – utility‑first framework, fully customizable

  • Vanilla JavaScript – lightbox, accordion, filters, scroll effects
  • Lucide Icons – consistent, lightweight SVG icons
  • Font Awesome 6 – social & brand icons
  • AOS (Animate on Scroll) – scroll‑triggered reveal animations
  • Google Fonts – elegant serif display + clean sans body

🎨 Customization

Change Brand Colors

Open css/styles.css and edit the :root block. Every component references these variables:

:root { --primary: #B4835B; /* main brand color */ --primary-dark: #8B5A3C; --secondary: #C9A27C; /* warm gold */ --accent-c: #F2EAE0; /* light neutral bg */ --muted-c: #6B5E53; /* text muted */ --gradient-primary: linear-gradient(135deg, var(--primary), var(--secondary)); --gradient-hero: linear-gradient(to bottom, rgba(255,255,255,0.7), rgba(255,255,255,0.3)); --shadow-luxe: 0 25px 50px -12px rgba(0,0,0,0.08); /* ... and more */ }

Change any value and the entire site updates instantly.

Typography

LuxeNest uses a serif display font (loaded as font-display in Tailwind config) and a clean sans‑serif body. You can replace them by editing the Google Fonts link in each HTML file and updating the fontFamily in the Tailwind config block.

Tailwind Configuration

Each HTML file contains a small tailwind.config script that extends the default theme with custom colors, fonts, and spacing. You can add or modify values there — no need to touch the CDN.

tailwind.config = { theme: { extend: { colors: { primary: 'var(--primary)', 'primary-c': 'var(--primary)', 'primary-dark': 'var(--primary-dark)', secondary: 'var(--secondary)', 'accent-c': 'var(--accent-c)', 'muted-c': 'var(--muted-c)', }, fontFamily: { display: ['"Cormorant Garamond"', 'serif'], }, } } }

📬 Forms & Interactions

The contact form (contact.html) and tracking form (order-tracking.html) use simple JavaScript to prevent default submission and display a success message. You can connect them to any backend:

  • Formspree – add your endpoint to the action attribute
  • Netlify Forms – add netlify attribute to the form
  • PHP Mail – point to your mailer.php
  • Zapier / Make – webhook integration
// Example: contact form success handler form.addEventListener('submit', function(e) { e.preventDefault(); // your submission logic here alert('Thank you — we will be in touch.'); this.reset(); });

Other interactions — accordion, lightbox, navbar glass effect, scroll‑to‑top — are fully functional out of the box and do not require any API keys.
